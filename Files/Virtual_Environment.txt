#Virtual Environments is an isolates dependencies for different Projects
#To create and use a virtual environment (venv) , Follow these steps

#1-For Create a venv execute this command on example "python -m venv myenv"

#2-To active the virtual environment execute this command on ex "source myenv/bin/activate"


#3-Install the dependencies inside the VENV execute this command "pip install numpy"

#4- to deactivate the venv execute this command "deactivate"



############### THE MAIN REASONS TO USE TH VENV ############

#1- PREVENT DEPENDENCIES CONFLICTS BETWEEN PROJECTS
#2- KEEP THE PROJECT DEPENDENCIES SEPARATES
